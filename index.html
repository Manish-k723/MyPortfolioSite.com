Autralia Rain Forecasting

This project aims to forecast tomorrow's rain in Australia using historical weather data. It involves exploratory data analysis, preprocessing, and feature engineering to prepare the data for machine learning training. Multiple algorithms such as linear regression, decision tree, random forest, and GRU are implemented to predict the weather. Evaluation metrics like mean squared error and r-squared values are used to assess the performance of the models and it forecasted with 85% accuracy on test-data.
